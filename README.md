# 📋 FocusList - Java To-Do Application

A simple console-based To-Do List application built using Java. This project allows users to manage their daily tasks efficiently with options to add, view, mark as completed, and delete tasks.

## 🚀 Features
- Add new tasks
- View all tasks
- Mark tasks as completed ✔️
- Delete tasks
- User-friendly menu system

## 🛠️ Technologies Used
- Java
- ArrayList
- Object-Oriented Programming (OOP)

## ▶️ How to Run

1. Compile the program:
   javac TodoApp.java

2. Run the program:
   java TodoApp

## 💡 Project Description

This project demonstrates the use of Java fundamentals such as classes, objects, loops, and conditional statements. It also focuses on improving user interaction by displaying tasks before performing actions.
